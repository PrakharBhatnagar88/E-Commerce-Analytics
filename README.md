This project provides a comprehensive 360-degree view of an e-commerce business's performance. By integrating data from multiple sources, I’ve built a dual-platform solution (Power BI & Tableau) to track high-level KPIs, customer behavior, and regional sales trends.


## Project Overview
The goal was to transform raw, fragmented sales data into actionable business intelligence. This project addresses critical business questions:

- Which product categories drive the highest profit margins?
- How do seasonal trends impact monthly sales volume?
- Where are the geographic hotspots for customer acquisition?


## The Toolkit
I chose a "best-of-breed" approach, using each tool for its specific strength in the data lifecycle:

- **Excel:** Initial data exploration and quick-fix cleaning (handling nulls and formatting).
- **SQL:** The "Heavy Lifter"—used for complex joins, CTEs, and aggregating raw data into clean, analysis-ready tables.
- **Power BI:** Built for deep-dive interactive reporting and DAX-driven custom metrics.
- **Tableau:** Leveraged for advanced geospatial visualization and high-performance storytelling.


## The Workflow
**1. Data Extraction:** Pulled raw transactional data from CSV/Database sources.

**2. SQL Processing:** Performed data transformation including:
   - Calculating Year-over-Year (YoY) growth.
   - Segmenting customers based on purchase frequency.
   - Standardizing currency and date formats.
     
**3. Modeling:** Established relationships (Star Schema) to ensure high-performance filtering in the dashboards.
     
**4. Visualization:** Designed intuitive layouts focused on UX (User Experience), ensuring stakeholders can find answers in under 10 seconds.


## Key Insights
- **Peak Seasonality:** Sales consistently spike in Q4, driven primarily by the "Electronics" and "Fashion" categories.
- **Shipping Efficiency:** Identified a correlation between "Standard Class" shipping delays and lower customer satisfaction scores in specific regions.
- **Profit Leaders:** While the West region has the highest volume, the East region maintains a 12% higher profit margin per order.
- **Retention:** 30% of total revenue is generated by a core group of "Repeat Customers," suggesting a need for a dedicated loyalty program.


## Dashboard Features
### Power BI Version
- **Dynamic Slicers:** Filter the entire report by Date, Region, and Category with a single click.
- **KPI Scorecards:** Instant visibility into Total Sales, Profit, and Quantity with YoY indicators.
- **Drill-Throughs:** Allow users to click a region and see specific customer-level transaction details.

### Tableau Version
- **Geospatial Maps:** Heatmaps indicating sales density across various states/provinces.
- **Trend Forecasting:** Built-in predictive analytics to estimate sales for the next quarter.
- **Interactive Tooltips:** Hover-over details that provide deep context without cluttering the visual space.


## How to Use This Repo
- **SQL Scripts:** Check the /SQL folder to see the logic used for data cleaning.
- **Dashboards:** Open the .pbix file for Power BI or the .twbx file for Tableau (links to Tableau Public are also provided in the documentation).
